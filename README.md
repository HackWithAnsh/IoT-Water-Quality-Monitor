# IoT Water Quality Monitor

An IoT-based water quality monitoring system using ESP32 and a turbidity sensor to monitor water clarity in real time.

---

## Project Overview
This project uses a turbidity sensor with ESP32 to detect the clarity of water. The sensor data can be monitored locally or uploaded to an IoT dashboard for remote monitoring.

---

## Features
- Real-time turbidity monitoring
- ESP32 Wi-Fi support
- Sensor-based water quality detection
- IoT dashboard compatibility
- Compact and low-cost design

---

## Components Used
- ESP32
- Turbidity Sensor
- Breadboard
- Jumper Wires
- USB Power Supply

---

## Working Principle
The turbidity sensor measures the cloudiness level of water. The ESP32 reads sensor values and processes the data. The output can be displayed on serial monitor or sent to an IoT platform for live monitoring.
